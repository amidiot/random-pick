# 2025 한슬 상품 추첨 프로그램

30명의 인원에게 30개의 상품을 랜덤으로 배정하는 제비뽑기 웹 애플리케이션입니다.

## 기능

- **수동 진행 방식**: 버튼 클릭마다 한 명씩 상품 공개
- **등급별 Confetti 효과**: 1등상 > 2등상 > 3등상 > 일반상품 순서로 화려한 효과
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원
- **결과 화면**: 등급별로 구분된 전체 당첨자 목록

## 상품 구성

- **1등상**: 에어팟 (1개)
- **2등상**: 인터불고 식사 2매 (1개)
- **3등상**: 술 (1개)
- **일반상품**: 가습기 (5개), 빵세트 (5개), 홍삼 (6개), 스파클링 와인 (6개), 베스킨 케잌 (5개)

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 엽니다
2. "추첨 시작" 버튼을 클릭합니다
3. "다음 추첨" 버튼을 클릭하여 한 명씩 상품을 공개합니다
4. 모든 추첨이 완료되면 결과 화면에서 전체 당첨자 목록을 확인할 수 있습니다

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
